# aws-s3-lab
This repository contains a step-by-step demonstration of how to create two S3 buckets and upload objects using the AWS Management Console.
It also includes the task explanation, lab steps file, and a YouTube video walkthrough.

Project Overview

Amazon S3 is a scalable object storage service used for storing files, images, documents, backups, and more.
In this lab, we perform the following tasks:

Create two S3 buckets

Upload files (objects) to each bucket

Record the process and upload the video to YouTube

Upload steps documentation to GitHub
